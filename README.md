# Timi · 私密日记

基于 **Flask + HTML5 + CSS3 + JavaScript** 的本地网页日记应用，界面参考 Notion 风格。服务仅绑定 `127.0.0.1`，数据保存在本机 `data/` 目录，**不上传网络**，适合个人私密记录。

当前版本：**UI v8**

---

## 功能总览

### 隐私与安全

| 能力 | 说明 |
|------|------|
| 纯本地运行 | 仅监听 `127.0.0.1`，不对外网暴露 |
| 可选访问密码 | 首次可设密或跳过；PBKDF2 校验密码 |
| 正文加密 | 启用密码后，日记内容以 Fernet 加密写入 `diaries.json` |
| 会话解锁 | 登录后在本机会话中解密编辑，关闭浏览器需重新解锁 |

### 界面与布局

| 能力 | 说明 |
|------|------|
| Notion 式布局 | 左侧边栏 + 顶部封面 + 主内容区 |
| 自适应宽度 | 主区域随窗口变宽（最高约 1600px），宽屏下两侧留白更少 |
| 浅 / 深色主题 | 侧栏一键切换，偏好写入 `meta.json` |
| 专注书写页 | 主区域以块编辑器为主；日期、心情、标签移至 **⚙ 页面设置** 侧栏 |

### 左侧「我的日记」

| 能力 | 说明 |
|------|------|
| 按月分组 | 日记按 `YYYY-MM` 折叠展示，点击展开/收起 |
| 文件夹 | 侧栏 📁 可新建文件夹，日记可归入文件夹 |
| 子页面 | 每篇旁 **+** 可创建子页面（层级 `parent_id`） |
| 按标签筛选 | 见下方「标签」说明；显示每标签篇数，点击筛选 / 再点取消 |

### 块编辑器（书写视图）

在书写页输入 **`/`** 唤起「基础块列表」，类似 Notion 块编辑：

| 块类型 | 说明 |
|--------|------|
| 文本 | 普通段落 |
| 待办 | 复选框 + 文字 |
| 标题 | H1～H4 |
| 列表 / 数字列表 | 条目列表 |
| 折叠列表 | 可展开标题 + 子块 |
| 引述 / 着重 | 引用与强调样式 |
| 代码 | 等宽代码块 |
| 图片 | 本地选图（建议 &lt; 2MB，Base64 存入） |
| 分割线 | 横线分隔 |

**编辑操作：**

- `Enter`：在文本类块下新建下一块  
- `Backspace`（空块）：删除该块  
- `⋮⋮`：拖拽调整块顺序  
- `/` 菜单支持键盘 ↑↓、Enter 选择  
- 底部快捷条可快速插入常用块  

**存储：** 正文以 JSON 块结构保存（`format: blocks`），同时生成纯文本供搜索、列表预览；旧版纯文本日记会自动转为「一个文本块」。

### 多视图（数据库）

| 视图 | 说明 |
|------|------|
| ▦ 表格 | 名称、心情、标签、日期、字数；支持关键词 / 标签 / 心情筛选与分页 |
| ▢ 看板 | 按 **心情** 分列展示卡片 |
| 📅 日历 | 月历显示有日记的日期；点击打开或在该日新建 |
| ✏️ 书写 | 块编辑器全屏写作 |

### 标签与筛选

1. 在 **页面设置** 中为日记添加标签（输入后回车，或点标签库）。  
2. 左侧 **「按标签筛选」** 显示各标签及篇数。  
3. 点击某标签 → 只显示该标签的日记（与顶部「全部标签」下拉联动）。  
4. 再次点击同一标签或点 **「清除标签筛选」** → 恢复全部。  

### 搜索、模板与导出

| 能力 | 说明 |
|------|------|
| 快速搜索 | 工具栏「搜索」或 **Ctrl+K**，搜标题与正文 |
| 日记模板 | 每日日记、周复盘、空白页，一键插入预设块 |
| 导出 TXT | 单篇或全部 |
| 导出 Markdown | 单篇 `.md`（块结构转 Markdown） |

### 统计

侧栏与书写页显示：**总字数**、**日记篇数**、**书写天数**、**本篇实时字数**。

## 快速开始

### 环境要求

- Python 3.10+（推荐）
- 依赖见 `requirements.txt`：`flask`、`cryptography`

### 安装与启动

```bash
# 进入项目目录
cd w:\项目\riji\-timi

# 安装依赖（建议使用虚拟环境）
pip install -r requirements.txt

# 启动服务（终端需保持运行）
python app.py
```

启动后终端会提示：

```
Timi 日记本 UI v8
请在浏览器打开: http://127.0.0.1:5000
```

浏览器访问：**http://127.0.0.1:5000**

> 若界面样式异常或仍是旧版，请在浏览器按 **Ctrl+F5** 强制刷新。侧栏底部可查看当前 UI 版本号。

### 首次使用

1. 可选择 **「启用加密保护」** 设置密码，或 **「暂不设置」** 直接进入。  
2. 侧栏 **+** 或顶部 **「新建」** 创建日记。  
3. 切换到 **✏️ 书写**，用块编辑器写作，点 **保存**。  
4. 心情、日期、标签等点顶部 **⚙ 页面设置** 配置。

## 使用说明（常用操作）

| 你想做的事 | 操作 |
|------------|------|
| 写一篇新日记 | 侧栏 `+` 或「新建」→ 书写 → 保存 |
| 编辑已有日记 | 侧栏或表格中点击该篇 |
| 插入标题、待办等 | 书写区输入 `/` 或点块类型快捷按钮 |
| 设置心情、标签 | **⚙ 页面设置** |
| 按标签查看 | 左侧「按标签筛选」点击标签 |
| 建文件夹 | 侧栏「我的日记」旁 📁 |
| 建子页面 | 侧栏某篇旁的 `+` |
| 切换表格/看板/日历 | 顶栏视图 Tab |
| 全文搜索 | Ctrl+K |
| 用模板 | 顶栏「模板」 |
| 导出备份 | 侧栏「导出全部」或书写页 TXT / MD |
| 锁定 | 侧栏「锁定」（需重新输入密码） |

**保存逻辑：**

- 无选中已有篇时保存 → **新建一篇**，保存后清空编辑区继续写。  
- 从侧栏/表格打开某篇后保存 → **更新该篇**。

## 数据存储

| 文件 | 内容 |
|------|------|
| `data/meta.json` | 密码相关元数据、标签库、文件夹列表、主题 |
| `data/diaries.json` | 所有日记条目（标题、日期、心情、标签、加密正文等） |

单条日记主要字段：`id`、`date`、`title`、`mood`、`tags`、`parent_id`、`folder_id`、`word_count`；正文为块 JSON 或历史纯文本。

## 隐私说明

- 服务 **只监听本机** `127.0.0.1`，不会在局域网或公网暴露（除非你自行改配置）。  
- **不使用** 第三方云存储、CDN 字体或行为统计。  
- 启用密码后，磁盘上的日记正文为密文；请牢记密码，遗忘无法找回。  
- 图片块以 Base64 存在 JSON 中，体积较大时请控制数量与大小。

## 常见问题

**终端里的 `WARNING: This is a development server...` 是报错吗？**  
不是。表示 Flask 内置开发服务器已启动，可忽略。停止服务请在终端按 `Ctrl+C`。

**保存后只有一篇 / 被覆盖？**  
新建保存后会进入「新篇模式」；要编辑旧篇请先从侧栏点开该篇再保存。

**界面很窄？**  
v8 已做自适应加宽；请强刷缓存并确认侧栏显示 UI v8。
